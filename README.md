# NBA Analytics Tableau Dashboard

This project was created for my IS440 Business Data Visualization and Communication final project. The goal of the project was to use Tableau to analyze NBA team records, player leaders, team shooting stats, and a career comparison between Michael Jordan and LeBron James.

## Project Overview

The project uses NBA data from the 2024-2025 season and career data for Michael Jordan and LeBron James. I created three Tableau dashboards to answer different research questions about team success, player performance, and career comparisons.

## Dashboards

### Dashboard 1: NBA Leaders

This dashboard looks at the top teams and player leaders during the 2024-2025 NBA season.

It includes:
- Eastern and Western Conference win-loss records
- Top 5 points per game leaders
- Top 5 assists per game leaders
- Top 5 3-point field goals per game leaders
- Top 5 rebounds per game leaders
- Top 5 blocks per game leaders
- Top 5 steals per game leaders

Key finding: Oklahoma City had the best overall record, while Cleveland had the best record in the Eastern Conference. Nikola Jokic stood out because he appeared in multiple leader categories.

### Dashboard 2: NBA Teams

This dashboard compares NBA teams using a map and team filter action. When a team is selected on the map, the rest of the dashboard updates to show that team’s record, age distribution, and shooting stats.

It includes:
- NBA teams by division map
- Selected team win-loss record
- Player age distribution and average age
- 2-point shooting by player
- 3-point shooting by player
- Free throw shooting by player

Key finding: Milwaukee had a stronger regular season than Miami, finishing with 48 wins compared to Miami’s 37 wins.

### Dashboard 3: Michael Jordan vs. LeBron James

This dashboard compares the career data of Michael Jordan and LeBron James.

It includes:
- Average points and games played by season
- Home and away win-loss percentages
- Career comparison stats
- Player image

Key finding: Jordan had the higher scoring average, while LeBron had stronger all-around numbers in assists, rebounds, field goal percentage, and total games played.
